# Onboarding Captadores TuMatch

Onboarding oficial para certificar captadores de **TuMatch Inmobiliario SpA** — Academia de Corredores · 2026.

🔗 **Producción:** https://tumatch-captador-onboarding.vercel.app
🔐 **Vista admin:** https://tumatch-captador-onboarding.vercel.app/admin.html

## Contenido

Sitio estático (HTML + CSS + JS vanilla) que entrega:

- **10 módulos** con contenido, ejemplos, cards y herramientas
  1. Sé Captador de TuMATCH
  2. Comisiones 2026 · Referido y Captador
  3. Círculos · Caliente, Tibio y Frío
  4. Motivación y Constancia
  5. Ecosistema y Servicios TuMatch
  6. CRM · Embudo de Captación
  7. Acuerdos · Con y Sin Exclusividad
  8. Captar en Precio · Tasación
  9. Captación de Arrendatario ★
  10. Tu aliado Team Leader
- **Banco rotativo de 20 preguntas por módulo** (5 a rendir)
- **Prueba final** con banco de 50 preguntas (10 a rendir · 10/10 aprueba)
- **Bloqueo 10 min** al acumular 5 errores
- **Alternativas aleatorizadas** en cada render
- **Certificado oficial** descargable PNG con `html2canvas`
- **CTA WhatsApp** para activar perfil al terminar
- **Dark/light mode** responsive desktop y mobile

## Archivos

```

```
